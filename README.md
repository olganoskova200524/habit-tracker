# Habit Tracker (Django + DRF)

## Стек технологий
- Django
- Django REST Framework (DRF)
- JWT-аутентификация (SimpleJWT)
- PostgreSQL
- Celery + Redis
- Telegram-бот для напоминаний
- drf-spectacular (OpenAPI / Swagger)

## Локальный запуск (черновик)
- Poetry — управление зависимостями и виртуальным окружением
- PostgreSQL — база данных для локальной разработки
